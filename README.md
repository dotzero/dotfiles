# dotzero dotfiles

## Install dependencies

### Xcode Command Line Tools

```bash
xcode-select --install
```

### Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

### Dotfiles

```bash
git clone git@github.com:dotzero/dotfiles.git ~/.0
```

## Bootstrap

```bash
~/.0/bootstrap
~/.0/macos
```

## Update

```bash
selfupdate
```
